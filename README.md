# React Weather App 
### `npm start`

#Runs the app
Open http://localhost:3000 to view it in your browser.
The page will load

## Search
Type in a city
typing in the wrong location wil give you a "Location not found. Please try again" error

## Information
Displayes a weather card of the city you searched 
- Weather description
- Feels Like
- Humidity
- Pressure
- Wind speed
